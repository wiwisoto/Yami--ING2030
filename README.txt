Yami Web Prototipo v2

Cómo usar:
1. Abre index.html en un navegador.
2. Ingresa con usuario: yami y contraseña: hola.
3. Revisa Perfil, Álbum y la lámina activa Pollo a la plancha.

No se usó API externa. El sitio funciona con HTML, CSS, JavaScript y archivos locales.
Las calificaciones de recetas se guardan en localStorage del navegador.
